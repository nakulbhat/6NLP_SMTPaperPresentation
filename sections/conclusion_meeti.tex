\section{Conclusion}

\begin{frame}{Summary and Task Simplification}
\begin{itemize}
    \item 22 research groups from 20 institutions
    \item 7 commercial systems and Google's online statistical MT system
    \item This time the focus was completely on news articles
    \item The unfair advantage of domain-specific training data? Eliminated.
\end{itemize}
\end{frame}

\begin{frame}{Human Evaluation Insights}
\begin{itemize}
    \item Introduction of an editing task to measure comprehensibility
    \item Learnt a lot about human reliability
    \begin{itemize}
        \item Tried dropping first 50 learning judgements
        \item Tried removing judges with the worst agreement
    \end{itemize}
\end{itemize}
\end{frame}

\begin{frame}{Automatic Metric Performance}
\begin{itemize}
    \item ULC metric correlated best for translations into English
    \item TERp and wpBleu performed best for the other languages
    \item Sentence level prediction proved to be challenging
    \item Using system-level scores as prior probabilities improved sentence-level scoring
\end{itemize}
\end{frame}
